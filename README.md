# image-utils (WIP)
Utils for image operation in borwser.

## Features

- [x] Read file object as data uri
- [x] Compress image
- [x] Convert data uri to Blob/File object
- [ ] Auto fix image orientation
- [ ] Rotate image
- [ ] Crop image

## Usage

### Use npm

```bash
npm install image-utils --save
```

### Use `<script>`

```html
<script src="path/to/image-utils.js"></script>

<script>
    ImageUtils.readFile();
    ImageUtils.compress();
    // ...
    // ...
</script>
```
